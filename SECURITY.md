# Security Policy

## Supported versions

The `main` branch receives security updates. Older tags are not supported.

## Reporting a vulnerability

1. **Do not** open a public issue.
2. Email `security@english-ia.local` with a detailed report that includes:
   - A clear description of the issue and its impact.
   - Steps to reproduce.
   - Any proof-of-concept exploit or logs.
3. You will receive an acknowledgement within 3 business days.

We aim to provide a fix and coordinated disclosure plan within 30 days when possible. Thank you for helping keep English IA safe.
